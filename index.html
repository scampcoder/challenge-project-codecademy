Property Name Description
color Changes the color of the text selected. color: blue;
opacity Changes how transparent an element is. opacity: 0.5;
text-align Positions the text right, left or center. text-align: right;
font-size Changes the size of the text. font-size: 20px;
font-family Changes the typeface of the text. font-family: "Arial";
background Changes the color of the background. background: teal;
